# Welcome to CNC Work Tracker!

This is a simple Progressive Web App **(PWA)** written in HTML which I developed to track my work activities. This runs in a "sandboxed" and semi-stripped down version of your default browser on mobile which makes the PWA appear as if it were a native app. 


# How it works:

1. Enter your **Name**, **Job Order Number** and **Drawing Number**.
2. Choose a predefined work activity or enter a custom one and hit **"Start Job"**
3. When you want to start a new job, just enter the new details and hit start again. The app will automatically log the current time as both the end and start time of the previous and new job respectively.
4. You can repeat this as many times as you want. At the end of a day you can choose **"End working time"** in the **"Work activity"** drop-down menu to mark the end of the currently active job.
5. At the end of a week or however long your window is before you need to export the work log according to the predefined format, choose **"End cycle"**. The PWA will then give a popup which allows you to copy the work log to clipboard to use wherever else you require it.

## Ease of use on mobile:

**Android:**
Steps to add the PWA to your home-screen and take advantage of the native app feel:

For **Chrome**:
1. Open the [Link](https://sgtshepard.github.io/cnc-work-tracker/)
2. Go to the **Three dots menu** (Top right usually)
3. Tap **Add to home screen**
4. Either choose to **install** or **add as shortcut**.
With **Install**, any changes I make to the site will not reflect there until you reinstall it. It runs fully offline and never checks for any updates. 
**Add as shortcut** still runs offline but if you delete your browser cache or force refresh the page it will pull the new version from this GitHub page.

For **Firefox**:
1. Open the [Link](https://sgtshepard.github.io/cnc-work-tracker/)
2. Go to the **Three dots menu**
3. Tap **More**
4. Tap **Add to home screen**
5. Confirm **Add to home screen** through the Android dialogue box. 

>*I have no experience with Apple products, hence the lack of instructions. I assume the process is similar regardless*
